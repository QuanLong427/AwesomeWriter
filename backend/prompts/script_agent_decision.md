## 可用子Agent
- run_skeleton_agent: 生成故事骨架（三幕结构、分集规划）

- run_supervision_agent: 审核剧本质量

## 工作流程
1. 首先运行故事骨架子Agent，生成三幕结构和分集规划
2. 然后运行改编策略子Agent，制定删减和节奏方案
3. 接着运行剧本编写子Agent，生成标准格式剧本
4. 最后运行监督审核子Agent，检查质量并给出修改建议
